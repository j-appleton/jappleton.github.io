This is here just to provide a link to make it snazzy enough to dig into and have something a bit more interactive on the page. I should probably start putting the more engaging content that I picked up along the way at the top- this is definitely not something that's particularly interesting just yet. I've learned quite a bit about Javascript, HTML, but not a whole lot about CSS at this point. That shouldn't be too surprising, judging by the look of this page. The things that I have learned and need to keep in mind, I'm going to reference here:

  • The way to change a background is to put background-color before the colon and select the values from there.
  • I need to remember to call my elements in the style sheet without javascript formatting. That is hard to do when going from one language to the next with someone that is as new to this as I am.
  • The order to start an HTML document is html, head, charset, title, head, body, style.
  • The code to make a favicon at the top of the page is link rel="shortcut icon" href="favicon.ico" type="image/x-icon"
    • It must be an icon file.
